# ═══════════════════════════════════════════════════════════════════════════════
# SYSTEM DEPENDENCIES (install separately, NOT via pip):
# ───────────────────────────────────────────────────────────────────────────────
#   ffmpeg          — required by moviepy, ffmpeg-python, scenedetect
#                     Ubuntu: sudo apt install ffmpeg
#                     Mac:    brew install ffmpeg
#
#   imagemagick     — required by moviepy TextClip (text overlays)
#                     Ubuntu: sudo apt install imagemagick
#                     Mac:    brew install imagemagick
#                     ⚠ On Ubuntu, you may also need to edit ImageMagick policy.xml
#                       to allow PDF/text: change <policy domain="coder" rights="none" pattern="PDF">
#                       to rights="read|write"
#
#   libsndfile1     — required by librosa / soundfile
#                     Ubuntu: sudo apt install libsndfile1
#
# ═══════════════════════════════════════════════════════════════════════════════
# MODELS DOWNLOADED AUTOMATICALLY ON FIRST RUN (sizes):
# ───────────────────────────────────────────────────────────────────────────────
#   Whisper base              ~150 MB   (via faster-whisper: auto-download)
#   Whisper small             ~480 MB
#   Whisper medium            ~1.5 GB   (recommended for accuracy)
#   BLIP-2 (blip2-opt-2.7b)  ~10 GB    (GPU strongly recommended)
#   emotion model             ~300 MB   (j-hartmann/emotion-english-distilroberta-base)
#   sentence-transformers     ~90 MB    (all-MiniLM-L6-v2)
#   Phi-3-mini (if no ollama) ~7 GB
#   DeepFace Facenet512       ~90 MB    (auto-download by deepface)
#   Coqui TTS model           ~200 MB   (auto-download on first TTS call)
#
# ═══════════════════════════════════════════════════════════════════════════════
# KNOWN ISSUES:
# ───────────────────────────────────────────────────────────────────────────────
#   1. moviepy TextClip failures: if you see "MoviePy error: creation of None"
#      → install imagemagick AND check its policy.xml allows text rendering
#      → OR run with --skip-text-overlays flag (falls back to ffmpeg drawtext)
#
#   2. BLIP-2 on CPU is extremely slow (~2-5 min/scene → days for 2h movie)
#      → Use --skip-captions on CPU-only machines; add captions separately on GPU
#
#   3. TTS install conflict: if `pip install TTS` fails due to numpy/torch conflicts
#      → Try: pip install TTS --no-deps, then install deps manually
#      → OR switch to kokoro-onnx (pip install kokoro-onnx)
#
#   4. deepface first run downloads multiple models (~500MB total)
#      → Expected on first use; subsequent runs use cache (~/.deepface/)
#
#   5. faster-whisper needs: pip install faster-whisper
#      The transcript_processor.py currently uses openai-whisper API.
#      If using faster-whisper, update WHISPER_BACKEND=faster in .env
